# latex速查

## 常用希腊字母
|小写	|大写	|latex |
|---|---|---|
|α	|A	|\alpha|
|β|	B	|\beta|
|γ	|Γ	|\gamma|
|δ	|Δ	|\delta|
|ϵ	|E	|\epsilon|
|ζ	|Z	| \zeta |
|ν	|N|	\nu|
|ξ	|Ξ|	\xi|
|ο	|O	|\omicron|
|π	|Π	|\pi|
|ρ	|P|	\rho|
|σ|	Σ	|\sigma|
|η	|H	|\eta|
|θ	|Θ	|\theta|
|ι	|I|	\iota|
|κ|	K|	\kappa|
|λ|	Λ|	\lambda|
|μ|	M|	\mu|
|τ|	T|	\tau|
|υ|	Υ|	\upsilon|
|ϕ|	Φ|	\phi|
|χ|	X|	\chi|
|ψ|	Ψ|	\psi|
|ω|	Ω|	\omega|

> 如何想使用小写字母latex代码开头字母需要为小写，如\alpha；使用大写字母latex代码开头字母需要为大写，如\Alpha。
